# 날씨 기반 인근 음식점 추천 웹사이트

오늘의 음식 웹사이트는 현재 장소의 날씨에 맞는 음식과 음식점을 추천해주는 웹사이트입니다.

## 제작자

- 이경엽 : 프론트엔드

- 김세영 : 백엔드

- 고지안 : 백엔드(5월 합류)

- 김민아 : 백엔드 및 디자인(5월 작업 종료)

## 접속

https://adoring-heyrovsky-d172e9.netlify.app/

## 기능

1. 날씨에 맞는 오늘의 음식 추천
2. 인근의 오늘의 음식점 추천
